# Visualising Forecasts with Excel

- Excel is a powerful tool for exploratory data analysis and visualization.
- Time series data can be effectively analyzed using line charts and Sparklines.
- Pivot tables help structure raw data for analysis.
- Forecasting can be done using the GROWTH function.
- Standard deviation and range help measure volatility.
- Scatter plots and trendlines help visualize relationships between variables.
- Correlation analysis quantifies how strongly two variables move together.
- Correlation matrices provide a complete view of relationships across multiple variables.
- Conditional formatting enhances interpretation of data patterns.
- Visualization helps identify trends, outliers, and hidden relationships quickly.
